# 总结Python的一些奇技淫巧

#### 链式比较操作
```bash
>>> x=8
>>> 1 < x < 10
True
>>> 10 < x < 20
False
>>> x < 10 <x*10 < 100
True
>>> 10 > x <= 9
True
>>> 5 == x > 4
False
```

你可能认为它执行的过程是 `1 < x`, 返回`True`，然后在比较 `True < 10`， 结果返回`True`，因为解释器会把`True`解析成`1`，`False`解析成`0`。但这里的链式比较解析器运行原理则不是这样的，底层会把这些链式比较操作转换成：`1 < x and x < 10`
